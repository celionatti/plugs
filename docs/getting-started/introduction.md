# Introduction

Welcome to **Plugs**, a modern, high-performance PHP framework designed for developers who value speed, security, and a modular "Plug-and-Play" philosophy. 

Plugs combines the best patterns from industry leaders with a unique focus on **Single Page Application (SPA)** integration and a robust **Security Shield**.

---

## The Core Philosophy

### 1. Zero-Config (Smart Defaults)
Plugs is designed to work out of the box. Most components (Mail, Cache, Database) come with pre-configured "Smart Defaults." You only create configuration files when you need deep customization.

### 2. Plug-and-Play Modularity
Built as a collection of core modules, Plugs allows you to enable or disable features like AI, Billing, or Queues dynamically. This keeps your application lean and fast.

### 3. Progressive Frontend
With the built-in **SPA Bridge** and **Reactive Components**, you can build dynamic, app-like experiences using standard PHP and HTML, without the overhead of heavy JavaScript frameworks.

---

## Key Framework Pillars

### ⚡ Performance
- **Fiber-Ready Concurrency**: Built-in support for PHP Fibers for non-blocking I/O.
- **OpCache JIT Optimization**: Automated OpCache warming for maximum execution speed.
- **View Streaming**: Stream responses to the browser for faster "Time to First Byte."

### 🛡️ Security
- **Security Shield**: An intelligent middleware layer that detects and blocks malicious request patterns.
- **Native CSP & Nonces**: Automatic Content Security Policy management for your views.
- **Encrypted Payloads**: All background/async data is encrypted by default.

### 🤖 Intelligence
- **AI Awareness**: Framework-level integration for AI Agents, Query Assistants, and automated content generation.
- **Smart Relationships**: An ORM that automatically discovers and manages database relationships.
