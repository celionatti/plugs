# Introduction

Plugs is a modern PHP framework that prioritizes developer experience, security, and performance. It provides a familiar syntax for developers coming from other major PHP frameworks while remaining lightweight and flexible.

## Why Plugs?

- **Modular Design**: Use only what you need.
- **SPA Ready**: Built-in support for Inertia-like Single Page Application features.
- **Security First**: Comprehensive protection against CSRF, SQL Injection, and DDoS attacks.
- **Clean Syntax**: Intuitive API for routing, database operations, and view rendering.
- **Powerful CLI**: Automate repetitive tasks with the `theplugs` command-line tool.

## Key Features

### Robust Routing
Define routes with support for parameters, middleware, and grouped routing.

### View Engine
A powerful templating engine supporting components, slots, and caching.

### Database Abstraction
A fluent Query Builder and Eloquent-like Model system for seamless database interaction.

### Security Shield
Advanced middleware that monitors request patterns to protect your application from malicious activity.
